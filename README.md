# WhatsApp Message Automation using Python

## Project Description

This project automates WhatsApp message sending using Python. It reads contact details from an Excel file and sends messages automatically through WhatsApp Web.

## Features

- Read contacts from Excel
- Send WhatsApp messages automatically
- Simple GUI using Tkinter
- Logging of sent messages
- Exception handling

## Technologies Used

- Python
- Pandas
- Tkinter
- PyWhatKit
- OpenPyXL
- Microsoft Excel
